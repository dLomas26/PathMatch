# AI Resume Analyzer

An AI-powered tool to analyze resumes and match them with job descriptions.  
Built with **React**, **JavaScript**, **Tailwind CSS**, and **Puter.js**.

---

## 🚀 Features
- Upload and store resumes securely
- AI-driven candidate-job matching
- ATS scoring & feedback
- Clean, responsive UI

---

## ⚙️ Tech Stack
- React + TypeScript
- Tailwind CSS
- Zustand (state management)
- Puter.js SDK (auth, storage, AI APIs)

---

## 🤸 Quick Start
```bash
git clone https://github.com/dLomas26/PathMatch.git
cd PathMatch
npm install
npm run dev
```
